# TomAwesome Modal Notices

A responsive and customizable modal notice plugin for WordPress.

TomAwesome Modal Notices lets website owners create unlimited modal windows, banners, announcements, and notices without using an external service.

## Features

- Page-load, delayed, scroll, exit-intent, and click triggers
- Sitewide, page, post, post-type, taxonomy, and archive targeting
- Optional start and end scheduling
- Visitor frequency controls
- Center, banner, side, and corner placement
- Custom colors, dimensions, typography, and animations
- Accessible keyboard and focus management
- No advertising, telemetry, or external-service requests

## Requirements

- WordPress 5.9 or newer
- Tested through WordPress 7.0
- PHP 7.4 or newer

## Installation

### Install from WordPress.org

1. In WordPress, open **Plugins → Add New Plugin**.
2. Search for **TomAwesome Modal Notices**.
3. Click **Install Now**, then **Activate**.
4. Open **Modal Notices → Add New**.

The plugin is also available from the [official WordPress.org Plugin Directory](https://wordpress.org/plugins/tomawesome-modal-notices/).

### Manual installation

1. Download the installable ZIP from the GitHub Releases page.
2. In WordPress, open **Plugins → Add New Plugin → Upload Plugin**.
3. Select the ZIP and install it.
4. Activate **TomAwesome Modal Notices**.
5. Open **Modal Notices → Add New**.

## Privacy

The plugin does not collect or transmit personal information.

Frequency controls use the visitor's browser `sessionStorage` or `localStorage` to remember the modal ID and display time. This information remains inside the visitor's browser.
