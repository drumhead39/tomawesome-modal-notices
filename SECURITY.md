# Security Policy

## Supported Versions

Only the latest released version of TomAwesome Modal Notices receives security updates.

| Version | Supported |
| ------- | --------- |
| 1.1.x   | Yes       |
| Earlier versions | No |

## Reporting a Vulnerability

Please do not disclose suspected security vulnerabilities through a public GitHub Issue.

Instead, use this repository's private vulnerability reporting feature:

1. Open the **Security** section of this repository.
2. Select **Advisories**.
3. Click **Report a vulnerability**.
4. Describe the issue, its potential impact, and how it can be reproduced.

When possible, include:

- The affected plugin version
- WordPress and PHP versions
- Steps to reproduce the problem
- Any proof-of-concept code or screenshots
- A suggested correction, if known

Please allow reasonable time to investigate and release a correction before publicly disclosing the vulnerability.
